RobotScheduler

多 Agent · 调度 · 路径规划

基于大模型与多 Agent 协作的大规模机器人智能调度系统，用于解决仓储场景中的死锁、路径冲突与动态障碍响应慢等问题。

  • 核心痛点：多机器人协作易死锁与冲突、全局效率低
  • 核心逻辑流：感知变化 → 评估周围 Agent 路径 → 预测拥堵概率 → 动态生成全局最优避障路由（冲突率显著降低）

lianghua

LLM · 量化 · 投研

基于大模型分析与长链决策的自动化量化交易 Agent 矩阵，面向财报/新闻等非结构化信息的理解与交易执行闭环。

  • 核心痛点：传统量化难理解非结构化文本、策略滞后
  • 核心逻辑流：数据抓取 → 情感解析 → 长链推导（数据变动→产业影响→情绪发酵→量价异动）→ 执行 Agent 生成买卖信号

codeSafety

安全审计 · 红蓝对抗 · 语义

基于 LLM 深度语义理解的自动化代码漏洞挖掘 Agent，引入红蓝对抗协作，强调发现业务逻辑漏洞与跨文件越权。

  • 核心痛点：传统 AST/规则扫描误报高、难发现复杂逻辑漏洞
  • 核心逻辑流：攻击者 Agent 推演攻击链（入口污染→跨文件调用追踪→数据泄露判定）→ 防御者 Agent 生成精准修复 PR
打开仓库 →

easyEarn

推荐匹配 · 风控 · 反欺诈

基于用户画像推理的自动化网赚任务匹配与反欺诈 Agent 平台，通过风控与匹配双 Agent 协作提升审核效率与任务质量。

  • 核心痛点：高价值任务匹配难、黑产/诱导任务多、人工审核成本高
  • 核心逻辑流：风控 Agent 深度语义解构任务 → 匹配 Agent 多步推理用户意图与偏好标签 → 千人千面分发（审核效率显著提升）

MakeMoneySmallAPP

小程序后端 · 增长运营 · 数据推理

由生成式 AI 驱动的智能变现与私域运营小程序后端引擎，内置自动化运营多 Agent 协作系统，面向用户留存与内容/策略迭代。

  • 核心痛点：内容同质化、人工运营僵化、留存率低难覆盖获客成本
  • 核心逻辑流：多 Agent 基于用户埋点数据进行长链推理与策略生成
